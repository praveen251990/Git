Prerequisties for Starting the application on Windows
1.Install Ruby
2.Install Rails
3.Install Devkit for native gems on windows.
Starting the Webserver
*In this i'm using the default webserver WEBrik and sqlite*
Download the source code from github and cd to the root the project.
Execute rails s or rails server to start the application.
Navigate to localhost:3000/welcome/index to view to page.

Added travis.yml and database.yml to test the app with different DB
